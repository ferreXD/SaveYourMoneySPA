Para lanzar el servidor desde la máquina virtual:

    1. Abrir el Visual Studio en la MV
    2. Seleccionar el proyecto reciente
    3. Lanzar la aplicación haciendo click en el botón "IIS Express"

Tras seguir esos pasos se debería de abrir un navegador web que mostrará la página de login.

Otra opción es directamente abrir alguno de los navegadores web de la MV ya que la página 
por defecto es la URL proporcionada del servidor Azure donde está alojada la aplicación.
O bien abrir su navegador personal e introducir esa URL (recomendado).
